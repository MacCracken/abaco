# Code of Conduct

## Our Standards

- Welcoming, inclusive language
- Respect for diverse viewpoints and experience
- Constructive feedback acceptance
- Focus on what is best for the community

## Scope

Applies to all project spaces: issues, pull requests, and discussions.

## Enforcement

Report concerns to conduct@agnos.dev.
